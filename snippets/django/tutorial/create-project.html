We would be building a blog in Django which would basically cover:

  • Using contrib.adminto create fully-featured admin pages
  • Adding tags to posts easily using taggit app
  • Class-based generic views for rapidly building pages (New in Django 1.3)
  • Using template inheritance and filters
  • Leveraging contrib.syndicationfor a simple feed

Let's start the project by installing the latest version of Django

pip install django

This would install the latest version of Django (i.e 1.10)

Letâ€™s start with our project now ....

  1. Start the project

    django-admin startproject djblog

  2. Browse the project and run the below commands to create a superuser (admin rights) and migrations.

     
    
cd djblog

./manage.py migrate (Does all the migrations for the in-built apps auth, sessions etc. )

./manage.py createsuperuser (Creates a super user with admin rights. Provide your username and password to create sucessfully)

./manage.py runserver (Run the development server and browse 127.0.0.1 to check the Django powered web-page)
  3. Add a new application blog

    ./manage.py startapp blog

  4. Open the project in your convenient Editor (PyCharm, Sublime Text 3 etc. )
  5. Edit djblog/settings.py to add "blog" in the list of installed apps.

    INSTALLED_APPS = [
    'django.contrib.admin',
    'django.contrib.auth',
    'django.contrib.contenttypes',
    'django.contrib.sessions',
    'django.contrib.messages',
    'django.contrib.staticfiles',
    'blog',
]
  6. Add the Template search directory path in djblog/settings.py

     
    
TEMPLATES = [
    {
        'BACKEND': 'django.template.backends.django.DjangoTemplates',
        'DIRS': [os.path.join(BASE_DIR, 'templates')],
        'APP_DIRS': True,
        'OPTIONS': {
            'context_processors': [
                'django.template.context_processors.debug',
                'django.template.context_processors.request',
                'django.contrib.auth.context_processors.auth',
                'django.contrib.messages.context_processors.messages',
            ],
        },
    },
]
