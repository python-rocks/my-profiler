Prerequisites and Installation

Let's make sure we have the following installed and ready to go.

Requirements

Python 2.7 or higher

Installation through pip in a Virtualenv


The most efficient way to install Django on your system is using the virtualenv tool. This tool allows you to create virtual Python environments where you can install any Python packages you want without affecting the rest of the system. This allows you to select Python packages on a per-project basis regardless of conflicts with other project's requirements.

Install pip :

sudo apt-get install python-pip (Python2)
sudo apt-get install python3-pip (Python3)

Once pip is installed, use it to install the virtualenv package.

sudo pip install virtualenv (Python2)
sudo pip3 install virtualenv (Python3)

Now create a your new project directory:

mkdir ~/myFirstProject
cd myFirstProject

Now, create a virtual environment within the project directory by typing:

virtualenv djangoenv

This will install a standalone version of Python, as well as pip, into an isolated directory structure within your project directory.
Activate this isolated environment to install packages using:

source djangoenv/bin/activate

Your prompt should change to reflect that you are now in your virtual environment. It will look something like

(djangoenv)username@hostname:~/myFirstProject$.



In your new environment, you can use pip to install Django in your virtual environment.

pip install django

Once installed, verify the version in the Python interpreter -

>>> import django
>>> django.VERSION

To exit your virtual environment, type "deactivate" from anywhere in the system.
