Installing and Launching PyCharm on Linux

Download

Download Pycharm Professional / Community version from here

Installation

Unpack the .tar.gz archive that you've downloaded, into any desired installation location.

  1. Unpack the pycharm-community-2017.2.4.tar.gz file to a folder (/opt/ recommended).
    
$ sudo tar xf pycharm-community-2017.2.4.tar.gz -C /opt/

  2. Switch to the bin directory: 
    
$ cd opt/pycharm-community-2017.2.4/bin

Launch PyCharm

Run pycharm.sh from the bin subdirectory.


$ ./pycharm.sh
