Installing and Launching Sublime Text 3 on Linux

Download

						
							$ wget http://c758482.r82.cf2.rackcdn.com/sublime_text_3_build_3083_x64.tar.bz2

Installation

  1. Unpack the .tar.gz archive that you've downloaded, into any desired installation location. The whole process is described below: 
    								
									$ tar vxjf sublime_text_3_build_3083_x64.tar.bz2

  2. Now we should move the uncompressed files to an appropriate location.
    								
									$ sudo mv sublime_text_3 /opt/.

  3. Create a symbolic link to use at the command line: 
    								
									$ sudo ln -s /opt/sublime_text_3/sublime_text /usr/bin/sublime

Launch

Just type sublime on the terminal and you are ready to go

							
								$ sublime
